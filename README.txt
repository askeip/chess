Шахматы

Чтобы начать игру,нужно запустить main.py(для консольной графики) или pyQtGraphic.py(для графической версии). В данной версии игры присутствует искусственный интелект и доступна игра только на одном компьютере.

Консольная версия:
Поле представляет собой квадрат 8х8,слева и снизу находятся индексы полей.Белые фигуры обозначаются через "W",черные через "B".Далее следует имя фигуры:
"R" - ладья,"N" - конь,"B" - слон,"Q" - ферзь,"K" - король и "P" - пешка. Для того чтобы выбрать фигуру нужно ввести сначала нижнюю координату,потом боковую. В следующей строке вводится поле,на которые вы хотите сходить.Вопросы и баги можно прислать на paparazzi730@gmail.com
