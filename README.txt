Chess
Start console version by running main.py or run pyQtGraphic.py for version with graphics. You can play versus bot (which makes random moves) or versus other player on same pc.

Console version:
Gamefield shown as 8x8 square, left column and bottom row show indicies. White pieces shown as "W", black as "B". Next follows name of piece: "R" - rook, "N" - knight, "B" - bishop, "Q" - queen, "K" - king and "P" - pawn. To select piece you should first enter bottom coordinate then left. Next line you enter field, which you want to go to. For example 46 44 to make e2 e4 move. Questions and bugs can be sent to askeipx@gmail.com

Шахматы

Чтобы начать игру,нужно запустить main.py(для консольной графики) или pyQtGraphic.py(для графической версии). В данной версии игры присутствует искусственный интелект и доступна игра только на одном компьютере.

Консольная версия:
Поле представляет собой квадрат 8х8,слева и снизу находятся индексы полей.Белые фигуры обозначаются через "W",черные через "B".Далее следует имя фигуры:
"R" - ладья,"N" - конь,"B" - слон,"Q" - ферзь,"K" - король и "P" - пешка. Для того чтобы выбрать фигуру нужно ввести сначала нижнюю координату,потом боковую. В следующей строке вводится поле,на которые вы хотите сходить. Например 46 44, чтобы сделать ход e2 e4. Вопросы и баги можно прислать на askeipx@gmail.com
